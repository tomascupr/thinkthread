# CORT SDK

SDK for interacting with CORT services.

## Installation

```bash
pip install cort-sdk
```

Or with Poetry:

```bash
poetry add cort-sdk
```

## Usage

```python
import cort_sdk
```

## CLI Usage

```bash
python -m cort_sdk
```

## Development

```bash
# Install dependencies
poetry install

# Run tests
poetry run pytest
```
